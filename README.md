# Resume Sense

Resume Sense is a smart AI-powered resume analyzer designed to help job seekers optimize their resumes for Applicant Tracking Systems (ATS). By providing instant feedback, ATS scores, and improvement tips, it helps you get closer to your dream job.

## 🚀 What it does

- **Smart PDF Upload**: Easily drag and drop your PDF resume for analysis.
- **AI-Powered Feedback**: Uses Puter.js AI capabilities to analyze your resume against specific job descriptions.
- **ATS Scoring**: Get a realistic assessment of how your resume performs for a given job title.
- **Image Conversion**: Automatically converts PDF resumes to high-quality images for better AI processing.
- **Data Privacy**: Leverages Puter.js for secure file storage and key-value data management.

## 🛠 Tech Stack

- **Framework**: [React Router v7](https://reactrouter.com/)
- **AI & Backend Services**: [Puter.js](https://puter.com/) (Authentication, AI Chat, File System, KV Storage)
- **PDF Processing**: [PDF.js](https://mozilla.github.io/pdf.js/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **State Management**: [Zustand](https://zustand-demo.pmnd.rs/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Package Manager**: [npm](https://www.npmjs.com/)
## 💻 Development Setup

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS recommended)
- [npm](https://www.npmjs.com/)

### Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/taherlokhandwala/resume-sense.git
    cd resume-sense
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:5173`.

4.  **Build for production**:
    ```bash
    npm run build
    ```
